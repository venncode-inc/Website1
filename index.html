cara pasang sc di termux

install termux di android > lalu buka dan tunggu sampai loading selesai > ketik $apt update && apt upgrade > lalu install pkg yang diperlukan, misalnya $pkg install nodejs git ffmpeg libwebp imagemagick wget -y > lalu ketik $termux-setup-stforage dan klik izinkan > lalu ketik dimana lokasi sc berada. misalnya $cd /sdcard/v88 > lalu run dengan ketik $node index.js
